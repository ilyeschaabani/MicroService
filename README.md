Ce projet est une plateforme de gestion des projets réels pour les étudiants et les consultants. Il permet la soumission, validation, suivi et finalisation des projets, ainsi que l’accompagnement des étudiants pour leurs projets de fin d'études (PFE). La solution est divisée en deux parties :

Backend : Spring Boot avec une base de données H2 pour l'accompagnement PFE et MySQL pour le microservice gestion des projets.

Frontend : Angular pour l’interface utilisateur interactive et responsive. 

Clonez le repository et naviguez dans le dossier frontend :
               cd frontend
Installez les dépendances avec npm :
               npm install
Lancez le projet Angular :
               ng serve
